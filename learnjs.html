Chapter 1 - The Basics

JavaScript is very different from HTML in amny ways, mainly that it does not use <p> tags, or any others. However, JavaScript does use semicolons after every statement. So, to get started, create a blank HTML file like this:

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>JavaScript<Title>
  </head>
  <body>



  </body>
  </html>

To insert JavaScript into a HTML file, use the <script> and </script> tags, like this:

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>JavaScript<Title>
  </head>
  <body>
    <script>

    </script>
  </body>
</html>

Chapter 2 - The Language

JS is very different from HTML. For example, instead of using <tags>, it uses a ; after every statement. Some of the most frequently used statements are:

if () {
};

while () {
};

text();

rect();

var;

and many more. For now, we will be creating a simple pop-up. But, in order to use commands such as text(); rect(); and others, we need a canvas to draw on and the ProcessingJS library. So, at the beginning of each program, add

  <script src="processing-1.6.6.min.js"></script>
</head>
<body>
  <p align="center">
    <canvas id="canvas"></canvas>
  </p>

(The </head> and <body> tags are just to show where the code is, but they are already there.) And, at the end, add
