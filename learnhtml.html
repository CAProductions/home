A step to step guide on HTML web page design

When you first start off with a new website add this base to it


<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>Example</title>
</head>
<body>

</body>
</html>

It might not make much sense right now but it will soon.

Lesson 1: The <p> tag

The <P> tag is one of the greatest in the website designers toolbox, in fact you are reading text created by it right now!
If you haven't guessed already the <p> tag makes a paragraph on the webpage, the "<" at the beginning and end of it signal
the computer that it is a tag, if you don't use them then your code will not work.
Lets add one to our code right now, put a <p> tag right below your <body> tag:


<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>Example</title>
</head>
<body>
<p>
</body>
</html>


now put any text after it.
hopefully it worked but now another issue arises "how do I create line breaks?" You can't just press enter because the computer
doesn't recognize that. In this case we need:

Lesson 2: The <br> tag

If you want to create a line break in your text you could either:

  • Use a </p> to end the paragraph (more on that later)
  • Use the <br> tag to create a line break
  • Give up and face defeat

In my personal opinion I would use the <br> tag but ending the paragraph and starting a new one also works.
here is an example of how to use a line break:

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>Example</title>
</head>
<body>
<p>John slammed the door shut <br> He was so mad that he was beaten by a 3rd grader
</body>
</html>

here's what that would look like:

John slammed the door shut
he was so mad that he was beaten by a 3rd grader

Lesson 3: Ending tags

Remember how earlier I was talking about ending the paragraph and starting a new one instead of using a line break?
adding a "/" before the letters in a tag creates an ending tag, most tags need these to function properly
One of the only tags that doesnt is the <br> tag. This is useful for things such as adding headers and many other HTML
tags afterward. Having a ending tag can also make a new paragraph just like the <br> tag although it is much easier to use the
<br> tag.
here Is an example of when you could use an ending tag instead of a <br> tag:

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>Example</title>
</head>
<body>
<p>John slammed the door shut <br> he was so mad that he was beaten by a 3rd grader</p>
<p>"Are you okay up there?" His Mom called</p>
<p>"Doing great" John said sarcastically</p>
</body>
</html>

here's what it would look like:

John slammed the door shut
He was so mad that he was beaten by a 3rd grader

"Are you okay up there?" His Mom called

"Doing great" John said sarcastically

Lesson 4: Headers

So right now our text is kinda small, and say I want to make a book. I need a Title and a Chapter Title.
Lets add a Header to our Webpage. You can use the <h1> tag for that.
the 1 In <h1> is the size, 1 is the biggest 6 is the smallest. I'm going to add a title and a chapter to my code:

</body>
</html>

here's what it would look like:

Lesson 5: CSS

Now that you know how to get text on your screen you might want to start changing the colors of the background and text, right?
For that we are going to need CSS, First lets add a <style> tag underneath or <title> tag in our head:

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>Example</title>
<style>

</style>
</head>
<body>
<h1>Jhon and the speedy Kid</h1>
<h3>Chapter 1: The Loss</h3> <p>John slammed the door shut <br> he was so mad that he was beaten by a 3rd grader</p>
<p>"Are you okay up there?" His Mom called</p>
<p>"Doing great" John said sarcastically</p>
</body>
</html>

Lets say I want to make my background a nice paper white and my text black, what I am going to do is underneath style write: body{color:; background-color:;} make sure to use the curly brackets "{}", here's an example:

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>Example</title>
<style>
body{
color:rgb(0, 0, 0);
background-color:rgb(255, 255, 255);
}
</style>
</head>
<body>
<h1>Jhon and the speedy Kid</h1>
<h3>Chapter 1: The Loss</h3> <p>John slammed the door shut <br> He Was So Mad that He Was Beaten By a 3rd Grader</p>
<p>"Are you okay up There?" His Mom Called</p>
<p>"Doing Great" John said sarcastically</p>
</body>
</html>

Lesson 6: Id Selectors

if you only want to change the style of one paragraph or header you can use an id. If you were to use one in a paragraph the tag would look like: <p id="p1">. Then In your style section you can reference just that id using #youridname then all the stuff you would normally out after it here is an example:

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>Example</title>
<style>
#red{
color:rgb(255, 0, 0);
}
#green{
color:rgb(0, 255, 0);
}
#blue{
color:rgb(0, 0, 255);
}
</style>
</head>
<body>
<p id="red">This text Is Red</p>
<p id="green">This text Is Green</p>
<p id="blue">This text Is Blue</p>
</body>
</html>

This Is what that would look like:

This text Is Red

This text Is Green

This text Is Blue

Lesson 7: Other CSS Tags

Background-color and color aren't the only CSS Tags, here is a list of other useful tags for text:

  • text-size:10px;
  • text-decoration:underline;
  • text-align:center;
  • font-weight:bold;

Those are just a few of the other tags you could use
here are some ways to use them:

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>Example</title>
<style>
#font-size{
font-size:40px;
}
#text-decoration{
text-decoration:underline;
}
#text-align{
text-align:center;
}
#font-weight{
font-weight:bold;
}
</style>
</head>
<body>
<p id="font-size">font-size</p>
<p id="text-decoration">text-decoration</p>
<p id="text-align">text-align</p>
<p id="font-weight">font-weight</p>
</body>
</html>

here's what that would look like

font-size

text-decoration

text-align

font-weight

Lesson 8: Organized Lists

If you want to make a list there are 2 ways to do it, you can either use an organized list which is numbered :

  1. Nintendo
  2. Xbox
  3. Playstation

Or you could use an unorginized list which is not.
To start an organized list use the <ol> tag, and us <li> to make a item on the list.
Here is an example:

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>Example</title>
</head>
<body>
<p>Top 5 Gaming Consoles</p>
<ol>
<li>Xbox Series X</li>
<li>Nintendo Switch</li>
<li>Nintendo 3DS</li>
<li>Playstation 5</li>
<li>My Play And Watch Game Pad!!!!!</li>
</ol>
</body>
</html>

Heres what that would look like:

Top 5 Gaming Consoles

  1. Xbox Series X
  2. Nintendo Switch
  3. Nintendo 3DS
  4. Playstation 5
  5. My Play And Watch Game Pad!!!!!

Lesson 9: Unorginized Lists

Lesson 10: Images

To Add An Image Use The <img> tag. In side it put src="". So like this: <img src="">. then inside of the quotation marks put an image adress. Example:

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>Example</title>
</head>
<body>
<img src="https://caproductions.github.io/home/assets/html/Dog.png"></img>
</body>
</html>

Heres what that would look like:

Yay! it worked, but it is a little on the big side. to fix that to the left of src put width="#" and replace
the # with how wide it should be (the browser will automaticlly put the height to the correct amount so it isnt streached or squashed).
Here is an example:

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>Example</title>
</head>
<body>
<img src="https://caproductions.github.io/home/assets/html/Dog.png" width="100"></img>
</body>
</html>
